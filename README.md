# Appointment Scheduler

Планировщик встреч на React + TypeScript. Позволяет выбрать дату, время, заполнить контактные данные и записаться на приём. Все записи хранятся локально в браузере (localStorage).

## Стек
- React 18
- TypeScript
- Vite
- CSS

## Как запустить локально
```bash
npm install
npm run dev
